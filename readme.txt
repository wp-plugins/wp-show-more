Hide the text between a simple Shortcode.

== Description ==

This plugin hides the text that is defined between the simple shortcode.<br>By clicking on the user-defined link, the hidden text becomes visible.<br><br>
It works via a shortcode and can be placed everywhere!<br>Usage: <strong>[show_more title="Read more"] Your hidden Content [/show_more]</strong><br><br>
Feel free to try it <a target="_blank" href="http://www.jamos.ch/plugins/wp-show-more">HERE</a>

== Installation ==

All you have to do is upload the plugin within your Wordpress Plugins menu and activate it. 
Then, you can add the shortcode everywhere you want (pages, posts, widgets etc): 

<strong>[show_more title="Read more"] Your hidden Content [/show_more]</strong> 

Of course, you can customize the text "Read more".
